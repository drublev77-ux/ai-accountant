## Step 1: Push to GitHub

```bash
# Create repository on GitHub first: https://github.com/new
# Name it: ai-accountant

# Then run:
git remote add origin https://github.com/drublev77-ux/ai-accountant.git
git push -u origin main
```

## Step 2: Deploy to GitHub Pages

1. Go to: https://github.com/drublev77-ux/ai-accountant/settings/pages
2. Under "Source", select **GitHub Actions**
3. Done! Your site will be at: `https://drublev77-ux.github.io/ai-accountant/`

## Step 3: Deploy to Vercel

1. Go to: https://vercel.com/new
2. Import repository: `drublev77-ux/ai-accountant`
3. Click **Deploy**
4. Done! Your site will be at: `https://ai-accountant.vercel.app`
