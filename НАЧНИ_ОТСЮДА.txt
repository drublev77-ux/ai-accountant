  🔄 ВАРИАНТ 2: С GITHUB (автообновления)

  Время: 3-5 минут | Сложность: ⭐⭐
  С синхронизацией кода и автодеплоем

  ШАГИ:

    1. Получите GitHub токен:
       → https://github.com/settings/tokens/new
       → Права: repo (все) + workflow
       → Скопируйте токен

    2. Сохраните токен:
       echo "GITHUB_TOKEN=ваш_токен" > .env.local

    3. Запустите деплой:
       ./АВТОДЕПЛОЙ.sh

  Что произойдет:
    1. Создаст репозиторий на GitHub
    2. Загрузит весь код
    3. Настроит Vercel
    4. При каждом git push - автообновление!
