🔑 ВАЖНО - АУТЕНТИФИКАЦИЯ:

   GitHub не принимает пароли. Используйте:

   1. Personal Access Token:
      • Создайте: https://github.com/settings/tokens
      • Выберите: "Tokens (classic)" → "Generate new token"
      • Права: repo (полный доступ)
      • При git push используйте токен вместо пароля

   2. Или SSH:
      git remote set-url origin git@github.com:YOUR_USERNAME/YOUR_REPO.git
