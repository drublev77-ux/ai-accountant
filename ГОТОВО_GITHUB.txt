🚀 ЗАПУСТИТЬ АВТОМАТИЧЕСКУЮ НАСТРОЙКУ:
════════════════════════════════════════════════════════════════

ВАРИАНТ 1: Интерактивный (рекомендуется)
────────────────────────────────────────────────────────────────
bash auto-github-setup.sh

Что произойдет:
1. Установится GitHub CLI (если нужно)
2. Откроется браузер для авторизации
3. Вы войдете в GitHub
4. Все настроится автоматически!


ВАРИАНТ 2: С токеном (быстрее)
────────────────────────────────────────────────────────────────
bash github-setup-token.sh

Что нужно:
1. Получить токен: https://github.com/settings/tokens/new
2. Выбрать права: repo, workflow
3. Скопировать токен
4. Вставить в скрипт
