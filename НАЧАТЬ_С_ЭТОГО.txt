ВАРИАНТ 1: СИНХРОНИЗАЦИЯ ЧЕРЕЗ GITHUB (РЕКОМЕНДУЕТСЯ)
─────────────────────────────────────────────────────

1️⃣  Создайте репозиторий на GitHub:
   • Перейдите: https://github.com/new
   • Название: ai-accountant-app (или любое другое)
   • НЕ инициализируйте с README

2️⃣  Добавьте remote и синхронизируйте:
   
   git remote add origin https://github.com/ваш-username/ai-accountant-app.git
   npm run sync

3️⃣  (Опционально) Запустите автоматическую синхронизацию:
   
   npm run sync:watch

   Теперь все изменения будут автоматически пушиться в GitHub!


ВАРИАНТ 2: ПРЯМАЯ СИНХРОНИЗАЦИЯ С ANDROID STUDIO
─────────────────────────────────────────────────

1️⃣  Создайте конфигурацию:
   
   npm run sync:studio

2️⃣  Отредактируйте .sync-config.local:
   
   # Для локальной машины:
   SYNC_METHOD="rsync"
   LOCAL_ANDROID_PATH="/путь/к/AndroidStudioProjects/MyApp"

   # Для удаленной машины через SSH:
   SYNC_METHOD="rsync"
   REMOTE_ANDROID_PATH="user@192.168.1.100:/путь/к/проекту"

3️⃣  Запустите синхронизацию:
   
   npm run sync:studio
