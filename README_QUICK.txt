🔄 ВАРИАНТ 2: С GITHUB (с автообновлениями)

    1. Получите токен: https://github.com/settings/tokens/new
       Права: repo (все) + workflow

    2. Сохраните токен:
       echo "GITHUB_TOKEN=ваш_токен" > .env.local

    3. Запустите:
       ./АВТОДЕПЛОЙ.sh
