💡 НУЖНА ПОМОЩЬ С АУТЕНТИФИКАЦИЕЙ GITHUB?

При git push попросит пароль - используйте Personal Access Token:

1. https://github.com/settings/tokens
2. "Generate new token (classic)"
3. Выберите срок, дайте права "repo"
4. Скопируйте токен
5. Используйте как пароль
