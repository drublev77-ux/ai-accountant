# Быстрый старт: Автоматическая сборка Android AAB

## За 5 минут до первой автоматической сборки

### 1️⃣ Создайте Keystore (1 минута)

```bash
keytool -genkey -v -keystore release.keystore -alias ai-accountant -keyalg RSA -keysize 2048 -validity 10000
```

**Введите пароль и запомните его!** Например: `MyPassword123`

### 2️⃣ Конвертируйте в Base64 (30 секунд)

```bash
# Linux/Mac
cat release.keystore | base64 > release.keystore.base64

# Windows PowerShell
[Convert]::ToBase64String([IO.File]::ReadAllBytes("release.keystore")) > release.keystore.base64

# Скопируйте содержимое файла
cat release.keystore.base64
```

### 3️⃣ Настройте GitHub Secrets (2 минуты)

Откройте: **GitHub** → **Settings** → **Secrets and variables** → **Actions**

Добавьте 4 секрета:

| Secret | Значение |
|--------|----------|
| `ANDROID_KEYSTORE_BASE64` | Содержимое файла `.base64` |
| `ANDROID_KEYSTORE_PASSWORD` | Ваш пароль keystore |
| `ANDROID_KEY_PASSWORD` | Ваш пароль ключа (обычно такой же) |
| `ANDROID_KEY_ALIAS` | `ai-accountant` |

### 4️⃣ Загрузите на GitHub (1 минута)

```bash
# Создайте репозиторий на GitHub, затем:
git remote add origin https://github.com/ваш-пользователь/ваш-репозиторий.git
git push -u origin main
```

## ✅ Готово! Сборка запустится автоматически

Перейдите в **Actions** вкладку вашего репозитория и наблюдайте за процессом сборки.

## Скачать готовый AAB

1. Откройте **Actions** → выберите зеленый ✅ workflow
2. Прокрутите вниз до **Artifacts**
3. Скачайте **android-release-aab**

## Автоматические триггеры

- ✅ **Push в main/develop** → Release AAB
- ✅ **Pull Request** → Debug AAB
- ✅ **Ручной запуск** → Выберите debug или release

## Следующие шаги

1. Скачайте AAB из Artifacts
2. Загрузите в [Google Play Console](https://play.google.com/console)
3. Опубликуйте приложение
